FOREMAN **Agent — Vision** (`d:\UIpath\foreman-vision\main.py`, LangGraph, single `perceive` node) is built and verified end-to-end. Inputs `mediaPath`, `mediaType`, `assetId`. Pipeline: (1) download media from bucket `foremenbucket` folder `Shared/foremen v1` → Gemini `gemini-2.5-flash` (image=`Part.from_bytes`, video=inline `Blob` <20MB else Files API) with `_generate_with_retry` (backoff on 429/5xx — Gemini throws transient `503 UNAVAILABLE` "high demand") → strict-JSON `perception`; (2) read Data Fabric `AssetIssueHistory`; (3) deterministic `history_match` on the TOP fault; (4) guarded auto-log via `insert_record`.
